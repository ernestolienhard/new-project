# Supabase Portal

This is a simple React + TypeScript portal app that stores messages in Supabase and is deployable on Vercel.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env.local` file with:
   ```bash
   VITE_SUPABASE_URL=your-supabase-url
   VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
   ```
3. Run locally:
   ```bash
   npm run dev
   ```

## Supabase

Create a table named `portal_messages` with columns:
- `id` bigint generated by default as identity, primary key
- `created_at` timestamp with time zone default now()
- `content` text

## Deployment

Push this repo to GitHub, then connect it to Vercel. Vercel will build with `npm run build` and use the `dist` output.
